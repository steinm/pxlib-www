pxview: Program to read Paradox Database files

pxview is a program to read Paradox DB files. It is based on the pxlib library. pxview allows to output csv for reading with spread sheets or sql to import into a relational database. You can also just show some information and the table structure of the database.

Features

  • Reading Paradox DB files
