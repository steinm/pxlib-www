pxlib: C Library to read Paradox Database files

pxlib is a simply and still small C library to read and write Paradox DB files. It supports all versions starting from 3.0. It currently has a very limited set of functions like to open a DB file, read its header and read every single record. It can also extract blob data and write it to a file. The write support is still very experimental.

pxview is a small command line program which uses pxlib.

Features

  • Reading/writing Paradox DB files
  • Reading/writing of primary index files (.PX)
  • Recoding of character fields
  • Reading Blobs (.MB files) and write them to single files

Downloading

The most recent release of pxlib is version 0.2.5.

  • Source and Debian package of version 0.2.5 for PPC is available at sourceforge

To compile this you'll only need a C Compiler, a recent glibc with iconv or a not so recent glibc and librecode.
