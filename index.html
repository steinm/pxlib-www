pxlib: C Library to read Paradox Database files

pxlib is a simply and still small C library to read Paradox DB files. It supports all versions starting from 3.0. It currently has a very limited set of functions, which to open a DB file, read its header and read every single record.

Features

  • Reading Paradox DB files

Downloading
